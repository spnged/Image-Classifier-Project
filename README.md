## General info
The objective of this project is to perform as an image classifier and count the number of identifiable objects in an image. In this sample, we use a series of cells under a microscope as our test images. Using Python, otsu was utilized to determine the threshold in each image, sorting the image into trues and falses. 

In addtion to transforming the image into binary, water segmentation was used to differentiate between cells that were in close proximity.

## Further Applications
